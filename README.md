
## IntelliJ IDEA Chinese Language Pack（中文语言包）


把压缩包里的"resources_zh_CN.jar"拷贝到IDEA安装目录下的lib目录，重启即可！

主要是扫描PhpStorm的翻译，做的这个中文语言包，汉化程度有限……
